Reversing Chain-of-Thought

Work backwards to verify forward reasoning.

The Concept

Backward Verification

Reversing CoT verifies reasoning by working backwards from the conclusion to the premises. If an answer is correct, you should be able to trace back through the logic and arrive at the original conditions. Inconsistencies suggest errors.

The Principle

Valid reasoning should be reversible. If A leads to B leads to C, then starting from C should trace back to A.

Process

1) Solve forward to get answer. 2) Start from answer, work backward. 3) Check if you reach original problem. 4) If mismatch, revise.
