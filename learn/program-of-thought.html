Code-Based Reasoning

Program of Thoughts

Express reasoning as executable code for precise computation.

The Concept

Code as Reasoning

Program of Thoughts (PoT) separates reasoning from computation. Instead of having the LLM perform calculations in natural language, PoT generates executable code that represents the reasoning steps. An interpreter then runs the code to produce the final answer.

Why It Works

LLMs are good at understanding problems and writing code, but prone to arithmetic errors. By delegating computation to a real interpreter, PoT gets the best of both worlds.

The Process

1) Understand the problem. 2) Generate code that solves it. 3) Execute the code. 4) Return the computed result.

Comparison

CoT vs PoT

Chain-of-Thought

"A store has 128 apples. If 37 are sold each day for 3 days, how many remain?"

Reasoning: 37 Ã— 3 = 111 sold. 128 - 111 = 17 remain.

Program of Thoughts

initial = 128
sold_per_day = 37
days = 3
remaining = initial - (sold_per_day * days)
print(remaining) # Output: 17

When to Use PoT

Complex Math

Multi-step calculations, statistics, or iterative computations that benefit from precise execution.

Data Processing

Filtering, sorting, aggregating, or transforming structured data.

Symbolic Tasks

Pattern matching, string manipulation, or logical operations.
