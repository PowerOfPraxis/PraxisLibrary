DiVeRSe Prompting

Diverse paths plus verification for robust answers.

The Concept

Diversity + Verification

DiVeRSe (Diverse Verifier on Reasoning Steps) combines diverse reasoning paths with step-by-step verification. It generates multiple different solution approaches, verifies each step's correctness, and uses verification scores to select the best answer.

Three Components

1) Diverse prompts generate varied reasoning. 2) Verifier checks each step. 3) Voting weighted by verification scores.

Key Innovation

Unlike simple majority voting, answers from more verified chains count more heavily.

Process

DiVeRSe Pipeline

Diversify

Use different prompts and sampling to generate varied reasoning approaches.

Verify

Score each reasoning chain based on step-by-step correctness checks.

Select

Vote on final answer with weights from verification scores.
