Active Prompting

Target annotation effort where the model is most uncertain.

The Concept

Uncertainty-Guided Annotation

Active Prompting identifies questions where the model shows high uncertainty (disagreement across multiple samples), then prioritizes annotating chain-of-thought reasoning for those specific questions. This focuses human effort where it's most needed.

The Insight

High disagreement across samples indicates the model is unsure. These uncertain cases benefit most from human-annotated reasoning chains.

Four Steps

1) Sample multiple answers per question. 2) Measure disagreement. 3) Annotate CoT for high-uncertainty questions. 4) Use as demonstrations.

Process

Active Prompting Pipeline

1. Sample

Generate multiple answers for each question using temperature sampling.

2. Measure

Calculate uncertainty based on answer disagreement or entropy.

3. Annotate

Create chain-of-thought for the most uncertain questions.
