Prompt Mining

Automatically discover effective prompt templates.

The Concept

Automated Prompt Discovery

Prompt Mining automatically searches for effective prompt phrasings rather than relying on manual trial and error. By systematically exploring variations and measuring performance, it discovers templates that work well for specific tasks.

The Insight

Small phrasing changes can significantly impact performance. Mining explores this space systematically to find optimal formulations.

Process

1) Generate prompt variations. 2) Test on validation set. 3) Select best performers. 4) Refine and iterate.

Methods

Mining Approaches

Template Search

Try different sentence structures and phrasings for the same instruction.

Paraphrase Generation

Use models to generate paraphrases, then test which work best.

Gradient-Guided

Use optimization signals to guide prompt modification toward better performance.
