Ensemble Methods

Demonstration Ensembling

Combine multiple example sets for robust predictions.

The Concept

Ensemble over Examples

Demonstration Ensembling runs the same query with multiple different sets of few-shot examples, then combines the results. This reduces sensitivity to the specific choice of demonstrations and produces more reliable outputs.

Why It Matters

Few-shot performance can vary significantly based on which examples are chosen. Ensembling smooths out this variance by not relying on any single set.

The Process

1) Create N different example sets. 2) Run query with each set. 3) Aggregate results (vote, average, etc.).

Methods

Aggregation Strategies

Majority Vote

For classification, select the answer that appears most frequently across ensemble members.

Average

For numerical outputs, average the values or aggregate probability distributions.

Union/Intersect

For generation, combine unique outputs or find consensus elements.

Comparison

vs Self-Consistency

Self-Consistency

Same prompt, multiple samples via temperature. Varies the model's randomness.

Demo Ensembling

Different prompts (different examples), can use deterministic sampling. Varies the input context.
