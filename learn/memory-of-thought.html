Memory Systems

Memory of Thought

Learn from past reasoning to solve new problems.

The Concept

Persistent Reasoning Memory

Memory of Thought (MoT) stores successful reasoning chains from past problems and retrieves relevant ones when facing new problems. This allows the model to build on previous experience rather than starting fresh each time.

Key Idea

Past reasoning chains serve as a dynamic example bank, with retrieval based on similarity to the current problem.

Components

1) Memory store of past chains. 2) Retrieval mechanism. 3) Integration with current reasoning.

Process

MoT Pipeline

Store

Save reasoning chains from successfully solved problems to the memory bank.

Retrieve

Find similar past problems and their reasoning chains for the current query.

Apply

Use retrieved chains as context or inspiration for solving the new problem.
