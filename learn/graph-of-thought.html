Graph of Thoughts

Network-based reasoning with merging and feedback loops.

The Concept

Beyond Linear and Tree Structures

Graph of Thoughts (GoT) extends reasoning beyond chains and trees into arbitrary graph structures. Thoughts can branch, merge back together, form loops, and create complex networks that better model how ideas actually connect and combine.

Key Innovation

While trees only branch outward, graphs allow thoughts to converge. Multiple reasoning paths can merge their insights, and feedback loops can refine ideas iteratively.

Structure Comparison

Chain: A → B → C
Tree: A branches to B, C
Graph: A, B merge to C, C loops back

Operations

Graph Transformations

Generate

Create new thought nodes by expanding, refining, or exploring from existing nodes.

Aggregate

Merge multiple thought branches into a unified insight that combines their strengths.

Refine

Improve existing thoughts through feedback loops and iterative enhancement.

Example

GoT in Action

Problem

Sort a large list of numbers efficiently.

Graph Approach

Split: Divide list into 4 sub-lists (parallel branches)

Generate: Sort each sub-list independently

Aggregate: Merge sorted sub-lists pairwise

Aggregate again: Merge the two results into final sorted list
