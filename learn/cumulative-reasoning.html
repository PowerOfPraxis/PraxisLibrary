Iterative Reasoning

Cumulative Reasoning

Build knowledge progressively through accumulated propositions.

The Concept

Progressive Accumulation

Cumulative Reasoning iteratively generates propositions that build upon each other. Each step produces new facts or conclusions that get added to the working context, gradually constructing the knowledge needed to answer the question.

Key Feature

Unlike single-pass reasoning, each iteration explicitly adds to a growing knowledge base that informs subsequent steps.

The Loop

1) Generate new proposition. 2) Verify and add to context. 3) Check if answer derivable. 4) If not, repeat.

Components

Three Modules

Proposer

Generates new propositions based on the current context and accumulated knowledge.

Verifier

Checks whether proposed statements are valid and should be added to context.

Reporter

Determines when enough context exists to derive the final answer.
