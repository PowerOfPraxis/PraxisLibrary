# Icons Directory

This directory should contain SVG icon files used throughout the site.

## Required SVG Files

Place the following SVG files in the `SVG/` subdirectory:

- `linkedin.svg` - LinkedIn social icon
- `email.svg` - Email/contact icon
- `arrow-right.svg` - Right arrow for buttons
- `arrow-left.svg` - Left arrow for back navigation
- `home.svg` - Home icon for navigation

## Notes

- All icons should be SVG format for scalability
- Icons are styled via CSS filters (see styles.css for icon color classes)
- Default icon size is 18px × 18px (local-icon class)
- Edge nav icons are 24px × 24px
