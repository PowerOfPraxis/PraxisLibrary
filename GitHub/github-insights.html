CI/CD: GitHub to Hostinger Auto-Deployment

What It Shows: The complete deployment pipeline from GitHub repository to Hostinger hosting with auto-deployment enabled.
Why It Works: SSH Key Authentication: Secure connection between GitHub and Hostinger using deploy keys.

Branch-Based Deployment: Only the main branch triggers production deployments, protecting against accidental pushes.

Zero-Downtime Updates: Files are pulled and updated atomically, ensuring visitors never see a broken state.
Workflow: 1. Developer pushes to GitHub
2. Hostinger receives webhook
3. Auto-pull from repository
4. Live site updated instantly

Security-First HTML Architecture

What It Shows: The HTML head section of security.html with inline Content Security Policy meta tags.
Why It Works: Meta CSP Fallback: Even if server headers fail, the browser enforces security via meta tags.

Strict Referrer Policy: Prevents sensitive URL data from leaking to external sites.

Semantic HTML: Proper document structure with lang attribute for accessibility and SEO.
Security Headers: default-src 'none'

Code Architecture Highlights

No jQuery, no Bootstrap, no React. All functionality is built with vanilla HTML, CSS, and JavaScript for maximum performance and control.

Self-Hosted Assets

Fonts, icons, and images are all hosted locally. No CDN dependencies mean no third-party tracking and guaranteed availability.

Mobile-First Design

CSS written for mobile devices first, then enhanced for larger screens. This ensures optimal experience on all devices.

Technical Architecture Summary

Zero Runtime Dependencies

No npm packages, no CDN links, no framework overhead. Every line of code is authored and controlled, eliminating supply chain risks and ensuring predictable behavior.

Sub-Second Load Times

Largest Contentful Paint (LCP) under 700ms. Total Blocking Time (TBT) of 0ms. Cumulative Layout Shift (CLS) of 0. All Core Web Vitals in the green zone.

Enterprise Security Standards

Strict Content Security Policy, HSTS with preload, X-Frame-Options DENY, comprehensive Permissions-Policy. Every header configured for maximum protection.

Fully Responsive Design

Mobile-first CSS architecture with 4 breakpoints. Touch-optimized interactions. Consistent experience from 320px mobile to 4K desktop displays.

99.8% Image Optimization

Profile image reduced from 4.4MB to 7.1KB without visible quality loss. All images optimized for web delivery with appropriate formats and compression.

Automated Deployment

Git push triggers automatic deployment to production. No manual FTP uploads, no deployment scripts to maintain. Changes go live within seconds.
